mail-hook

Gmail to Discord summary relay

mail-hook is a personal productivity automation tool that watches a Gmail inbox, summarizes new messages, classifies them, and delivers a Discord notification for each email.

How it works

  • The app detects new Gmail inbox events.
  • It reads message metadata and message content needed to generate a summary.
  • It uses OpenAI to summarize and classify messages.
  • It sends a Discord embed notification for the user.
  • It stores limited processing state and delivery records in Google Cloud Firestore.

Who this app is for

This app is currently operated as a single-user personal automation for the Gmail account authorized by the app owner.
